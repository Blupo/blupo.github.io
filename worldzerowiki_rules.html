The Framework

This is not the policy. This is the framework that the policy is built on.

  1. Be nice.
  2. Don't be disrespectful.
  3. Don't be malicious.

The Policy

  1. This policy is not fixed, and can change over time.

  2. Please follow all channel-specific rules.

  3. Be nice. We want to create a friendly server with a good community.
    • Don't be annoying or obnoxious.
    • Don't start unnecessary arguments.
    • Remember every user is a person behind the screen.

  4. Don't be disrespectful. This includes:
    • Personal attacks or harassment against anyone
    • Responding to disrespectful behavior in kind
    • Using the guidelines to justify any disrespectful behavior

  5. Don't be malicious. This type of behavior is something we absolutely do not want in our server. This includes:
    • Publishing other members' personal information
    • Unsolicited advertising to members
    • Using alt accounts to evade moderation action and continue breaking rules
    • Using the guidelines as a weapon against other members

  6. This is a server about a Roblox game. As such we want to keep it friendly and safe for Roblox's audience.
    • Do not post NSFW/L or anything sexually suggestive
    • Don't use bad language
    • This includes usernames, nicknames, and profile pictures.
    • Trading any in-game currencies for any out-of-game currencies is not allowed.
